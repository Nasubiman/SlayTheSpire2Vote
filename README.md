# スレスパ2 / Slay the Spire 2 カード・レリック評価ランキング

[Slay the Spire 2](https://store.steampowered.com/app/1868140/Slay_the_Spire_2/) のカード・レリック・敵キャラを **S / A / B / C / D** の5段階で評価できる投票サイトです。

みんなの投票をリアルタイムで集計し、最強カードTier表を確認できます。

## 機能

- キャラクター別カード一覧（アイアンクラッド / サイレント / ディフェクト / ネクロバインダー / リージェント / 無色カード）
- カードタイプ（アタック / スキル / パワー）・レアリティでフィルタリング
- 強化前 / 強化後の画像切り替え
- 全レリック評価ランキング（スターター〜ショップレリックを網羅）
- 全敵キャラ強さランキング（通常 / エリート / ボス、エリア別フィルター）
- S〜Dの5段階評価で投票
- 投票結果をリアルタイム表示（分布バーで確認）
- 同一IPからの重複投票を防止（24時間で再投票可能）

## 技術スタック

| 役割 | 技術 |
|---|---|
| フロントエンド | Next.js (App Router) / Tailwind CSS |
| ホスティング | Vercel |
| データベース | Firebase Firestore |
| 画像配信 | Vercel Blob |
| 投票API | Next.js API Routes |
